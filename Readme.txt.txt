Project Name: Government Web Application

1. Description: My web app provides information to a potential client i.e. government entities of the projects we've worked on as well as who we are, and what we do. We are a small business venture that works on bidding government contracts so that we can provide a service to government entities. If the potential client has questions or want to learn more about our business they can email us directly for additional information. 

2. Features:
Responsive Design: The application is fully responsive and adjusts seamlessly for both desktop and mobile devices.
Authentication System: Users can register, log in, and log out securely.
Admin Panel: Allows users to manage or create content, manage groups, etc.

3. Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: Django
Database: SQLite
Other Tools: Bootstrap
